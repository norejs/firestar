# Firestar

## Demos
https://zt2.ztgame.com/act/anniversary2020/


## Install
```bash
yarn add "@norejs/firestar" --dev
```

## Usage
```javascript
import Firestar from "@norejs/firestar";
Firestar(container, count, duration, imgsrc);
```
#### Parameter
* container：
    * Type：DOM
    * Default: body
    * Description：火焰效果的容器
* count
    * Type：Number
    * Default: 30
    * Description：火焰效果的数量
* duration
    * Type：Number
    * Default: 5
    * Description：一个粒子从开始到结束的过度时间
* imgsrc
    * Type：String
    * Default: 一张火焰的图片
    * Description：粒子的图片
